.. function:: spikey(data)

   Calculates the "spikey-ness" of the histogram.

   Spikeyness is the ratio between the tallest bin and the average bin height.

   :param data: The 1-d data array

   :returns: Ratio of the tallest bin height and the average bin height.


.. function:: skewed(data)

   Calculates skew.

   Utilizes scipy.stats.

   :param data: The 1-d data array

   :returns: The data skew.
